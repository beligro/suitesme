# Landing Page для MNE IDET

## Структура проекта

```
frontend/src/components/Landing/
├── Landing.js                 # Основной компонент лендинга
├── Landing.module.css         # Стили основного компонента
├── Header/                    # Компонент шапки
│   ├── Header.js
│   └── Header.module.css
├── Hero/                      # Главная секция (в разработке)
├── HowItWorks/               # Секция "Как это работает" (в разработке)
├── Benefits/                 # Секция преимуществ (в разработке)
├── Results/                  # Секция результатов (в разработке)
├── FAQ/                      # Секция FAQ (в разработке)
└── Footer/                   # Подвал (в разработке)
```

## Что уже реализовано

### ✅ Header компонент
- Логотип MNEIDET
- Мобильное меню (гамбургер)
- Навигация для мобильных и десктопных устройств
- Кнопка "Войти"
- Адаптивный дизайн
- Фиксированное позиционирование
- Backdrop blur эффект

### ✅ Базовая структура
- Все компоненты созданы как заглушки
- CSS Modules подключены
- Маршрутизация настроена
- Шрифты подключены (Unbounded, Montserrat)

## Как тестировать

1. Запустите сервер разработки
2. Перейдите на `/landing` в браузере
3. Проверьте работу мобильного меню (кнопка гамбургер)
4. Проверьте адаптивность на разных размерах экрана

## Следующие шаги

1. **Hero секция** - главный экран с заголовком и CTA кнопкой
2. **HowItWorks секция** - с телефоном и описанием процесса
3. **Benefits секция** - карточки с преимуществами
4. **Results секция** - галерея фото клиентов
5. **FAQ секция** - аккордеон с вопросами
6. **Footer** - контакты и ссылки

## Технические детали

- **Mobile First** подход
- **CSS Modules** для изоляции стилей
- **CSS Variables** из Design System
- **Pixel Perfect** реализация по Figma
- **Адаптивная типографика** с clamp()
- **Smooth анимации** и переходы

## Design System

Основные цвета:
- Primary: #1B3C4D
- Secondary: #8296A6
- White: #FFFFFF
- Background: #F7F7F7

Шрифты:
- Заголовки: HeadingNow-96Bold (fallback: Unbounded)
- Основной текст: Montserrat
- Кнопки: Unbounded
