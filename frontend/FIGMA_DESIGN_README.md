# Новый дизайн главной страницы на основе Figma

## Обзор изменений

Главная страница была полностью переделана в соответствии с дизайном из Figma (MNEIDET). Новый дизайн включает:

### Основные изменения:

1. **Полноэкранный hero-раздел** с фоновыми изображениями
2. **Новая цветовая схема** в сине-серых тонах
3. **CSS Modules** для изоляции стилей
4. **Адаптивный дизайн** для мобильных устройств
5. **Встроенная навигация** в hero-секции
6. **Эффекты размытия** и градиенты

### Файловая структура:

```
frontend/src/
├── assets/images/           # Изображения из Figma
│   ├── hero-photo.jpg
│   ├── decorative-element-1.jpg
│   └── decorative-element-2.jpg
├── styles/
│   ├── variables.css        # Основные CSS переменные (не изменены)
│   └── figma-variables.css  # Переменные для Figma дизайна
├── components/
│   ├── HomePage.js          # Обновленный компонент главной страницы
│   ├── HomePage.module.css  # CSS модуль для главной страницы
│   ├── Layout.js           # Обновленный Layout (скрывает навигацию на главной)
│   └── Layout.css          # Обновленные стили Layout
└── public/
    └── index.html          # Добавлены Google Fonts
```

### Цветовая схема:

- **Основной темный**: `#1B3C4D`
- **Очень темный**: `#031C2D`
- **Светло-серо-голубой**: `#90A3AB`
- **Акцентный**: `#83989D`
- **Фон**: `#F7F7F7`
- **Белый текст**: `#FFFFFF`

### Шрифты:

- **Заголовки**: Unbounded (Google Fonts)
- **Основной текст**: Montserrat (Google Fonts)
- **Акцентный**: Unbounded (Google Fonts)

### Особенности реализации:

1. **CSS Modules**: Используются для изоляции стилей компонента
2. **Адаптивность**: Поддержка экранов от 480px до 1920px
3. **Эффекты**: Blur-эффекты, градиенты, тени
4. **Изображения**: Автоматически скачанные из Figma
5. **Навигация**: Встроена в hero-секцию, скрыта на других страницах

### Адаптивные брейкпоинты:

- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px
- **Mobile**: 480px - 767px
- **Small Mobile**: до 479px

### Как запустить:

1. Убедитесь, что все зависимости установлены:
   ```bash
   cd frontend
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm start
   ```

3. Откройте http://localhost:3000 в браузере

### Совместимость:

- ✅ Современные браузеры (Chrome, Firefox, Safari, Edge)
- ✅ Мобильные устройства
- ✅ Планшеты
- ✅ Различные разрешения экрана

### Примечания:

- Старые стили главной страницы сохранены в `HomePage.css` (не используются)
- Layout автоматически скрывает стандартную навигацию на главной странице
- Все изображения оптимизированы для веб-использования
- Дизайн полностью соответствует макету из Figma
